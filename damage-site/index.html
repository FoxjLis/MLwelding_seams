Технологии

В нашем проекте мы использовали:

Нейронная сеть на базе YOLO v8

YOLOv8 - это передовая модель обнаружения объектов в реальном времени, которая обеспечивает высокую точность распознавания объектов на фото. YOLOv8 обладает высокой скоростью и точностью работы.

Датасет

Для создания мы использовали датсет, который состоит из снимков. Мы аугментировали датасет и подобрали наилучшие параметры для обученияы

API

Наш сервис разработан с использованием технологии API что позволяет Вам объеденить его с вашим бизнесом или интегрировать в другой сервис

Yandex Cloud

Yandex Cloud - это бесплатный облачный сервис, который позволяет писать и выполнять код на Python прямо в браузере. Он предоставляет доступ к большому количеству вычислительных ресурсов, что делает его полезным для задач машинного обучения и анализа данных.
